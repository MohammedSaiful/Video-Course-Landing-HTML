# Tutor - Video Course Landing Page
- A modern, responsive, and high-conversion landing page designed for a video course platform. This project features a clean UI, smooth animations, and a dedicated contact system.

#  Live Demo
- View the live version of this project here: 
 Tutor Project   | [Live Demo](https://tutor-project-section-17and18.vercel.app/) 



## Features
- Fully Responsive: Optimized for desktops, tablets, and mobile devices using CSS Grid and Flexbox.
- Sticky Navigation: Interactive navbar that changes style and applies a glassmorphism effect on scroll.
- Mobile Menu: A slide-out navigation menu for seamless mobile browsing.
- Integrated Contact Form: Functional "Contact Us" page connected via Formspree for backend-less form handling.
- Dynamic UI Elements: Hover effects, scaled images, and a decorative SVG wave transition for a premium feel.

## Built With
- HTML5: Semantic structure for better accessibility and SEO.
- CSS3: Custom properties (variables), Grid, and Flexbox for layout and styling.
- JavaScript (Vanilla): Light logic for menu toggling and scroll effects.
- Font Awesome: For scalable vector icons.
- Google Fonts: Open Sans for clean typography.
